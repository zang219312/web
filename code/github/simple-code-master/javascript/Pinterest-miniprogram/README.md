## 数据

先说下我们的图片链接格式

所有的链接都是`http://cued.xunlei.com/demos/publ/img/P_${name}.jpg`这样的格式，我们需要改变name的值就行了，当`name`值小于10的时候，格式是`00x`,如`002`、`003`，大于10的时候就是`023`这种。
